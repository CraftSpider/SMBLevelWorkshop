Debug Shortcuts

Shortcut Default Dolphin Keyboard Keybinding Action
D-Pad Up T +100s to the timer
X and Y S and C Complete the stage
R and Start W and Return Debug pause
L and Start Q and Return Open new debug main menu window
B and X Z and C Modify the UI
Must be done while paused / debug paused
Use the Main Stick or the D-Pad to move the cursor
A (Default is X on keyboards) will pick up and drop UI components

Windowing System

Shortcut Default Dolphin Keyboard Keybinding Action
Control Stick Up / Left / Down / Right Move the window
C Stick I / J / K / L Resize the window
D-Pad Up / Down T / G Navigate through the menu
D-Pad Left / Right F / H Change setting
A X Select menu item
L and A Q and X Cycle between focused windows / unfocus all windows
L and B Q and Z Send window to front / back
L and Y Q and S Close window
