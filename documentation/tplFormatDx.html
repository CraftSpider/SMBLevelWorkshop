File header


Relative Offset (decimal) No. Bytes (hex) Description
0 0x04 The ascii characters "XTPL"
4 0x04 Number of Textures
Note: Repeat rest of header for each texture
0 0x04 Texture ID (ignored)
4 0x04 Texture offset
8 0x02 Width of texture (ignored)
10 0x02 Height of texture (ignored)
12 0x02 Number of mipmaps (ignored)
14 0x02 0x1234

Texture


Relative Offset (decimal) No. Bytes (hex) Description
0 0x04 Texture ID
4 0x04 Width
8 0x04 Height
12 0x04 Number of mipmaps
16 0x04 0 if "uncompressed", 1 if "compressed"
20 0x04 Texture data size if uncompressed (can be size of compressed data on occasion?)
24 0x02 A size/length of some kind (0 if uncompressed)
28 0x02 0 (Probably padding to make the header 32 bytes)
32 Varies Texture Data
